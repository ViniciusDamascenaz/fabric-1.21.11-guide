# Tick System

## O que é

O Minecraft funciona em um sistema de ticks.

- **1 tick = 0,05 segundo**
- **20 ticks = 1 segundo**

Basicamento todo o tempo no jogo é baseado nisso.

---

## Quando você vai usar

- Quando criar delays
- Criar eventos programados
- sincronizar eventos do jogo

## Observações

- É muito comum confundir durante o desenvolvimento do seu mod os ticks com segundo, isso pode acabar causando problemas em meio ao desenvolvimento, principalmente durante a criação de eventos pesados e lentos.
