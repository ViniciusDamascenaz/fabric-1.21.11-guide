# Scheduler

## O que é

Um scheduler é uma função essencial para o seu mod se você quiser criar funcionalidade que envolvam agendar eventos ou manipular ticks. Ele permite que determinado código só seja executado depois de um certo tempo(em ticks)

É essencial para:

- Criar Dialogos
- Cenas
- Eventos em sequência
